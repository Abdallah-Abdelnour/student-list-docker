---->docker registry
docker build -t student_list:v1 .

Tu tagues cette image pour ton registre local : docker tag student_list:v1 localhost:5000/student_list:v1
localhost:5000 → ton registre privé local
student_list:v1 → le nom + tag de ton image

Tu pousses l’image vers ton registre : docker push localhost:5000/student_list:v1

Tu peux tirer cette image depuis n’importe quelle machine qui a accès à ce registre : docker pull localhost:5000/student_list:v1

Visualiser ton registre:
Comme tu l’as fait, ton registre local fonctionne sur : http://localhost:5000
Mais Docker Desktop ne te montre pas les images stockées dans ce registre.
Pour ça, tu peux utiliser : 
CLI : curl http://localhost:5000/v2/_catalog
UI : lancer un front-end web pour Docker Registry comme je t’ai montré avant.
